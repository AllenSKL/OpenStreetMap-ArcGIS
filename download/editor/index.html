OpenStreetMap Editor for ArcGIS

It is vital that the correct version of the ArcGIS Editor for OSM is downloaded and installed. Without it none of the tools created by the GASC will work rendering the whole process detailed in the workflows useless.

Download

  • ArcGIS Editor for OSM, 10.3.x Desktop >> Download here.
  • ArcGIS Editor for OSM, 10.2.x Desktop >> Download here.
  • ArcGIS Editor for OSM, 10.1.x Desktop >> Although this is available for download on the Esri website the toolbox is out of date and will not work with the GASC tools. Please contact us and we will send over a customised version of the 10.3.x toolbox that works on 10.1.x.
