Troubleshooting

Background processing

There is a known issue with running several of the OpenStreetMap Editor for ArcGIS tools when background processing is enabled in ArcMap. The result will be ArcMap crashing or the tool not completing successfully, often with an 'invalid tool' error. To disable background processing, follow instructions in the Foreground and background processing ArcGIS 10.1 help topic.
